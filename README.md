# Центр имени Мирослава — лендинг

Прототип сайта на чистом HTML / CSS / JS. Без сборки и зависимостей —
открывается двойным кликом по `index.html`.

## Структура

```
miroslav-center/
├── index.html              # вся разметка (одностраничный лендинг с якорями)
└── assets/
    ├── css/
    │   ├── reset.css        # нормализация
    │   ├── variables.css    # дизайн-токены: палитра, шрифты, отступы
    │   └── styles.css       # все стили по секциям
    ├── js/main.js           # бургер-меню, scroll-reveal, видеогалерея, форма заявки
    ├── img/                 # logo.jpg (полный), logo-mark.jpg (знак для навбара), фото мастеров
    ├── videos/              # видеоролики для секции «Наши специалисты»
    └── fonts/               # сюда локальные шрифты, если уходить от Google Fonts
```

## Секции

Шапка → Hero → О центре (счётчики + цитата) → Мастера (5) →
Наши специалисты (видео) → Услуги (01–06) → Цитата основателя →
Запись + контакты → Футер.

### Наши специалисты (видео)

Галерея вертикальных роликов (9:16). Видео грузятся «лениво»
(`preload="metadata"` — тянутся только заголовки, а не весь файл),
полный ролик скачивается только по клику на золотую кнопку play.
При запуске одного ролика остальные ставятся на паузу.

- **Файлы** — `assets/videos/*.MP4` (подключены в `index.html`).
- **Подписи специалистов** — сейчас карточки без имён. Чтобы добавить
  имя/специализацию под роликом, вставьте после блока `.specialist-card__media`
  свой `<div class="specialist-card__body">…</div>` (стиль можно взять
  с `.master-card__name` / `.master-card__role`).
- **WordPress** — это нативные теги `<video>`, ровно как у блока «Видео».
  Карточка `.specialist-card` — самодостаточный повторяемый блок: удобно
  ложится на ACF-репитер или Gutenberg-паттерн.

## Что заменить перед запуском

- **Телефон** `+7 (000) 000-00-00` — в секции «Контакты» (`index.html`).
- **Логотип** — в навбаре стоит `logo-mark.jpg` (эмблема без текста, вырезана
  из `logo.jpg` через `sips`), в футере — полный `logo.jpg`. Внимание:
  в логотипе написано «**Академия**», а в тексте сайта везде «**Центр**» —
  определитесь с единым названием.
- **Фото мастеров** — сейчас инициалы в золотом круге; фото класть в `assets/img/`.
- **Форма** — фронт-валидация + экран «успеха». Бэкенда нет: в `main.js`
  (блок «Форма заявки») помечено место для `fetch()` на endpoint или
  отправки уведомления в Telegram.

## Дизайн

Палитра и шрифты выведены из шаблона клиента: кремовый фон, золото-акцент,
заголовки `Playfair Display`, текст `PT Serif` (оба с кириллицей).
Mobile-first, поддержка клавиатурной навигации и `prefers-reduced-motion`.
